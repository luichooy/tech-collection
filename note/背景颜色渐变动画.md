
背景渐变色从右向左运动，无限运动


```css
div {
  position: relative;
  width: 300px;
  height: 100px;
  margin: 20px auto;
  background: linear-gradient(to right, red, green, blue, red);
  background-size: 200% 100%;
  background-position: 0 0;
  animation: bgposition 2s infinite linear;
}

@keyframes bgposition {
  0% {
    background-position: 0 0;
  }
  100% {
    background-position: 200% 0;
  }
}

```
